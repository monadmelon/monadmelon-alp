## DIRECTORY STRUCTURE

projects/ - technology, systems, code
log/      - tech diary, dated posts, updates
usr/      - personal files, philosophy, and thoughts
aud/      - music, sound, audio explorations
net/      - external connections & links

## APPLICATIONS

To run the audio player, you can type 'audplayer' from any directory, or navigate to the 'aud' directory and type './audplayer'.

To run the notes app, you can type 'notes' from any directory.
